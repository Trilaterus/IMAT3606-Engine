Detailed Description

The ComponentCamera class essentially stores a 3D position and angle vector in order to equally offset all the objects affected. Consider OpenGL to be a fixed window frame, to give the illusion of moving a camera we move all the objects by an offset instead. This camera aims to provide a First Person Shooter (or FPS) style camera with strafe-like movement functions.
