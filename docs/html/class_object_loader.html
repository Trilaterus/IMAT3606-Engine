Detailed Description

The ObjectLoader class allows for the vertices, normals and texture normals to be parsed from varying layout of .obj files. Although called an ObjectLoader it in itself holds the values of the object (It could do with a name refactor). In this version the texture normals have been disabled to reduce risk of crashing.
